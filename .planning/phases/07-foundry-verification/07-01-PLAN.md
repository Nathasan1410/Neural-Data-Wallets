## Goal

Install Foundry toolchain, verify NeuralDataRegistry.sol compiles and passes tests, deploy to Base Sepolia testnet, and verify on BaseScan.

## Requirements Closed

- CONTRACT-01: NeuralDataRegistry contract deployed to testnet
- CONTRACT-06: Contract passes all Foundry tests

## Tasks

### 1. Install Foundry
- Run `curl -L https://foundry.paradigm.xyz | bash`
- Run `foundryup` to install latest version
- Verify with `forge --version`

### 2. Compile Contract
- Navigate to `contracts/` directory
- Run `forge build`
- Fix any compilation errors in `src/NeuralDataRegistry.sol`

### 3. Run Foundry Tests
- Run `forge test -vvv`
- Fix any failing tests in `test/NeuralDataRegistry.t.sol`
- Ensure 100% test coverage

### 4. Deploy to Base Sepolia
- Configure `foundry.toml` with Base Sepolia RPC
- Run deploy script: `forge script script/Deploy.s.sol --rpc-url https://sepolia.base.org --broadcast`
- Record deployed address

### 5. Verify on BaseScan
- Run `forge verify-contract` or use BaseScan UI
- Ensure contract source code is verified and public
