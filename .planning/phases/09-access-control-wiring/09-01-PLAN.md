## Goal

Connect the existing access control UI components to the smart contract so users can grant and revoke access via blockchain transactions.

## Requirements Closed

- ACCESS-01: User can grant access to researcher by wallet address
- ACCESS-02: User can revoke access from researcher address

## Integration Gaps Closed

- AccessControl UI → Contract `grantAccess()` / `revokeAccess()` calls
